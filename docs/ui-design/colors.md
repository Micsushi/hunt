# DESIGN: Colors

## Colors

Dark green surfaces with a single bright accent.

- **bg (#0f1a14):** Deep green-black. Page background.
- **panel (#172212):** Cards, nav bar, panels. One shade lighter than bg.
- **panel-strong (#1d2b18):** Elevated panels, input backgrounds.
- **panel-hover (#223320):** Row/card hover state.
- **text-primary (#d4e8d0):** Slightly green-tinted off-white. Readable, cohesive with the palette.
- **text-secondary (#7a9e80):** Labels, captions, metadata.
- **border (#2a3f2a):** Panel borders. Subtle - visible but not loud.
- **border-strong (#3a5a3a):** Focus rings, raised containers.
- **accent (#59a96a):** Muted mid-green. Primary actions, active nav, links. One color, one intent. Avoid neon/light green for large text areas.
- **accent-soft (#1e3a26):** Hover tints, selected row backgrounds.
- **accent-dim (#4f965f):** Secondary accent, less prominent actions.
- **warning (#f0b429) + warning-soft (#2d2410):** Amber. Auth expiry, retries.
- **danger (#f05252) + danger-soft (#2d1414):** Red. Errors, destructive actions.
- **positive (#9bdeac):** Success states. Use sparingly for status, not links.
- **info (#4db8d8) + info-soft (#122432):** Informational highlights.
