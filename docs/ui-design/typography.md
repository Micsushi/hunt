# DESIGN: Typography

## Typography

System font stack. Consolas for all technical data, IDs, and timestamps.

- **body-md:** Default text. 1rem / 1.5 line-height.
- **body-sm:** Secondary content, helper text, table cell secondary lines.
- **mono:** Job IDs, timestamps, URLs, log output. Always Consolas.

Section labels inside panels: 0.82rem uppercase, 600 weight, 0.08em letter-spacing, text-secondary - not large h2s.
