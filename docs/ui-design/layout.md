# DESIGN: Layout

## Layout

Top navigation bar (sticky) + full-width content area. No sidebar.

- Content max-width: 1280px, centred, 24px horizontal padding.
- Page sections stack vertically, 24px gap.
- Two-column grids: `repeat(2, 1fr)`, 20px gap.
- Three-column grids: `repeat(3, 1fr)`, 20px gap.
- Cards: 20px padding, `rounded.md` border-radius, 1px `border` colour, `shadow`.
- Row height in tables: 40px compact.
- Spacing scale: 4 / 8 / 12 / 16 / 20 / 24 / 32 / 48px only.
