# Templates

This folder contains editable starter files for Component 2 source material.

Current templates:
- `candidate_profile.template.md`
- `bullet_library.template.md`
- `ad_hoc_job_description.template.md`

Recommended usage:
- copy a template to a non-template file
- fill it with truthful data only
- keep it versioned in git if you want change history
